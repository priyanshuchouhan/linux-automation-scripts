# Linux Automation Scripts

## Objective 1 - System Health Monitoring Script

This script monitors system health by displaying:

* Disk Usage
* Memory Information
* Running Processes

Run:

```bash
./system_health_monitor.sh
```

## Objective 4 - Application Health Checker

This script checks application availability using HTTP status codes.

Run:

```bash
./app_health_checker.sh https://google.com
```

Output:

* Application Status: UP
* HTTP Status Code: 200

* Bash
